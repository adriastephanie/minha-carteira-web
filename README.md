# minha-carteira-web

Projeto que cria uma carteira com lançamento de contas a pagar e receber, além de dash para veriricar relatorio da cateira
## Features

- login
- lançamento de contas a pagar 
- lançamento de contas a receber
- dash

## Tech

## Installation

Use the package npm [pip](https://docs.npmjs.com/cli/v10/commands/npm-install) to install project

```bash
Clone the repository, install the necessary dependencies, and then run the application.

git clone https://github.com/repo
npm install
npm start
```
